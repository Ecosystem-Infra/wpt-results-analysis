Browser Specific Failures

These charts measure how many WPT tests fail in exactly one browser (of the three major engines - Chromium, Gecko, WebKit), or "Browser Specific Failures". Such tests are interesting as they mostly represent 'real' interop issues in agreed-upon web features.

(Click + drag on graphs to zoom, right click to un-zoom)

Stable browser versions

Results are from stable releases of browsers at the given date, with a minimal set of flags passed to enable testing.

wpt.fyi query

Notable dates:

  • Jan 23-29, 2020: Large numbers of referrer-policy/ tests added
  • ~March 18, 2020: Large numbers of referrer-policy/ tests combined into fewer files
  • April 8, 2020: Safari 13.1

Experimental browser versions

Results are from experimental releases of browsers at the given date, such as Safari Tech Preview. Where appropriate, experimental features are enabled (e.g. --enable-experimental-web-platform-features for Chrome.)

wpt.fyi query

Notable dates:

  • Oct 17, 2019: Safari Tech Preview 96
  • Dec 20, 2019: Safari Tech Preview 97
  • Jan 23-29, 2020: Large numbers of referrer-policy/ tests added
  • ~March 18, 2020: Large numbers of referrer-policy/ tests combined into fewer files
